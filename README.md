<h1>Network File Sharing in the Cloud (Azure)</h1>
This tutorial outlines the implementation of using Active Directory within Azure Virtual Machines to share files over the network.<br />

<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines/Compute)
- Remote Desktop
- Active Directory Domain Services

<h2>Operating Systems Used </h2>

- Windows Server 2022
- Windows 10 Pro

<h2>Prerequisites</h2>

- How to setup active directory in azure VM found here: https://github.com/jasonmolinet/configure-ad
